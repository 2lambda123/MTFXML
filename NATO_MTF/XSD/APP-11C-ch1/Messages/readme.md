This is the source directory for the XML Schema consolidation XSLT.  Expected files are 369 directories each containing messages.xsd, sets.xsd, composites.xsd, fields.xsd.

The directories may also include metadata cards: messages.xsd_nmrr_metadata_card, sets.xsd_nmrr_metadata_card, composites.xsd_nmrr_metadata_card.xml, and fields.xsd_nmrr_metadata_card.xml.
